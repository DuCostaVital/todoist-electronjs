<h3>Como rodar o projeto</h3>

```bash
#clonar repositório
$ git clone https://github.com/DuCostaVital/todoist-electronjs

# Entrar no repositório
$ cd todoist-electronjs

# instalar dependências 
$ npm install

# Iniciar aplicação
$ npm run start

```
